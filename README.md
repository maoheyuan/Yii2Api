# Yii2Api
Yii2Api 是为电商前台项目提供接口，方便前台项目的前后台分离


Yii2Api 主要有以下功能接口
---------------------
----购物车模块接口<br>
----支付模块接口<br>
----登录模块接口<br>
----首页统计模块接口<br>
----管理员管理模块接口<br>
----会员管理模块接口<br>
----商品模块接口<br>
----商品分类模块接口<br>
----订单管理模块接口<br>
----订单商品模块接口<br>
----地区管理模块接口<br>
----地址管理模块接口<br>
----轮播管理模块接口<br>
----常用设置模块接口<br>
----统计管理模块接口<br>
----我的中心模块接口<br>
